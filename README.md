# 🚀 i2TA – Instituto de Inteligência e Tecnologia Aplicada da Amazônia

**Deep Tech. Inteligência Aplicada. Impacto Industrial.**

---

## 📡 Sobre o Projeto

O **i2TA** é uma ICT (Instituição de Ciência e Tecnologia) de alta performance, posicionada na interseção entre **ciência de dados avançada** e as demandas reais da **Indústria 4.0**.

Este repositório contém o **Brandbook Digital** e o **Design System oficial**, concebidos sob uma estética **Big Tech**, com foco em:

* Autoridade institucional
* Precisão técnica
* Escalabilidade global

## 🚀 Como Utilizar

### 🔍 Visualização

Abra o arquivo `index.html` em qualquer navegador moderno para acessar o **Brandbook completo**.

---

### 🖼️ Exportação de Assets

Utilize o arquivo `og_image.html` para gerar imagens de compartilhamento social em formato **PNG (1200x630)**.

---

### 💻 Desenvolvimento

Utilize as classes baseadas em **Tailwind CSS** presentes no projeto para garantir:

* Consistência visual
* Escalabilidade
* Padrão de design unificado
